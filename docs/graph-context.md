# 🎭 Graph Context & Vision

This outlines the core conceptual philosophy underpinning the application.

## 🧩 CORE IDEA: “THE DEAL FLOOR”
Think of your interface as a living deal negotiation surface — like capital is physically moving, forming, and restructuring in real-time.

Not a standard layout.
It’s a fluid financial surface where relationships are “constructed” live.

## 1. VISUAL METAPHOR — “LIQUID CAPITAL INTERFACE”
Everything sits on a soft, responsive surface (like a pressure-sensitive table).
* When investors exist → they create impressions (like weight on a surface).
* Connections → tension lines between those impressions.
* The UI feels Elastic, Responsive, and Alive.

## 2. INVESTOR = “CAPITAL MASS TILE”
Forget circles.
**Shape:** Rounded rectangular tile (like a financial card), slight 3D elevation.
**Soft shadow:** indicates “capital weight”.

**Weight Effect:**
* Bigger investors = heavier tiles.
* Heavier tiles slightly sink into surface and pull nearby tiles closer.
* This creates organic clustering without forced manual layout grids.

## 3. CONNECTIONS = “TENSION BANDS”
No plain lines.
Use Elastic bands / tension strips.
* They stretch between tiles with slight curvature (like stress lines).
* Thickness = strength.
* When you drag a tile bands stretch and nearby tiles visibly react.

## 4. LAYOUT — “FREEFORM NEGOTIATION TABLE”
This is the key innovation: **No rigid layout.**
Tiles naturally settle depending on their relationships. 
*   **Sector Pull**: The physics engine parses each entity's primary `sector` tag and pulls it towards its defined `CLUSTER_CENTER`. This organizes the floor into thematic zones (e.g., SaaS, Fintech, AI) while allowing relationship ties to still influence final positioning.
*   **Manual Manipulation**: Users can drag investors, group them manually, and override the default gravity to create custom “deal clusters”.

## 5. DESIGN LANGUAGE
**Colors:**
* Base: matte black / graphite.
* Accent:
  * Electric green → capital flow
  * Soft gold → elite investors
  * Muted blue → passive
  
**Typography:** Tight, numeric-heavy. Feels like a Bloomberg Terminal x Notion AI hybrid.

You are NOT building a generic graph visualization tool.
You are building **a digital deal-making surface where capital has weight and relationships have tension**.
